Import a course from the "Win edition" of easygenerator to the Web:

Overview

What will be imported:

  • Course title
  • Chapters (as sections)
  • Pages with content in a chapter (as content items; NOTE: without its layout formatting)
  • Questions in a chapter (as questions in a correspondent section)
  • Supported question types:
    • Single choice
    • Multiple choice
    • Single choice image (NOTE: only one answer will be marked as the correct)
    • Fill in the blanks (NOTE: only one correct item in a dropdown will be marked as the correct)
    • Text matching
    • Statement
    • Hotspot question
    • Open question
  • Order of the sections will match the order of the imported chapters
  • Order of the questions and content items in a section will match the order of the imported slides/pages in an appropriate chapter
  • Question feedback
  • Supported content format: text with basic formatting, images

What will NOT be imported:

  • Video and audio files from the library
  • Glossary links
  • Navigation rules and internal links
  • Adaptive dashboard and learning objectives from
  • Question types: ranking text, ranking image, drag and drop image, likert (NOTE: they will be imported as content items with appropriate content)
  • Course preferences (mastery score, xAPI settings, language settings)
