You create more effecitive e-Learning as follows:

 

  • The author starts by creating a course outline and the learning objectives
  • Next to the actual content you create assessments that are tied to the learning objectives.
  • In the assessments you can create navigation rules like: 'when the learner scores 60% or more on this objective, navigate to that page or chapter next'. 
  • A combination of rules organizes the navigation.
  • You can set different percentages a learner should score on different objectives. When the established percentages have been reached for all objectives the course is complete.
